=======

**pyradix** is a Python wrapper and command line client for interfacing
with the Radix DLT public network.

* Free software: MIT license
* Documentation: https://pyradix.readthedocs.io.

Features
--------

* A Radix JSON-RPC API wrapper
* A CLI
